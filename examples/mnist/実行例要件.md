# 実行例要件: Fashion-MNIST

* 作成者: AI
* 目的: 実データを用いて、バイナリMLPのモデル構築、50エポックの学習、精度表示、学習済みモデル保存までの一連の使い方を提示する。
* 自動テストでは代替しない範囲: Fashion-MNIST全件のダウンロードと長時間学習。
* 実行方法: `go run github.com/sw965/crow/examples/mnist`
* 成功条件:
  * 初期精度と各エポックの精度が表示される。
  * 50エポックをエラーなく完了する。
  * `fashion_mnist_model.gob` が保存される。
* 副作用: ユーザーのホームディレクトリにデータセットキャッシュを作成し、カレントディレクトリのモデルファイルを置き換える。
